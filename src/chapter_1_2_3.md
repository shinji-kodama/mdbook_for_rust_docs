# タプル

複数の値の組みを表現するための型（要素の型は異なっていてもOK）

例

|タプル|要素の型|
|:-:|:-:|
|("left", "right")|("&str", "&str")|
|(0, 1)|("i32", "i32")|
|('0', 1)|("char", "i32")|
|('0', "1", (2, '3'))|("char", "i32", "(i32, char)")|
|(0,)|("i32",)|

要素にはピリオドでアクセス

```rust
fn main() {
    let t: (i32, &str) = (120, "文字列");

    let _t0 = t.0; // i32型の値
    let _t1 = t.1; // &str型の値
}
```
