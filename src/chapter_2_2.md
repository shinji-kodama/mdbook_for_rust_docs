# Rustを書いてみる
## Cargoについて
- Rustにおけるパッケージ管理兼ビルドツール

- `cargo *******` コマンドで処理を実行。

- `cargo —list`でコマンド一覧が表示できるので詳細はそちらを参照

### 代表的なコマンド

| コマンド | 内容 |
| --- | --- |
| new | プロジェクト新規作成 |
| check | コードチェック（ビルドはしない） |
| build | プロジェクトのビルド |
| run | プログラムの実行 |
| clean | ビルド生成物をすべて削除 |
| add | プロジェクトに依存関係を追加 |
| test | テスト実行 |
| fmt | コードフォーマット（rustfmtを実行） |
| clippy | 静的解析の実行 |
| help | コマンドのヘルプ |
